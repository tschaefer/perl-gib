=head1 Perl::Gib::Config

Singleton (https://metacpan.org/pod/MooseX::Singleton) configuration
object.

 use Perl::Gib::Config;
 
 Perl::Gib::Config->initialize(output_path => '/var/www/docs');
 
 my $perlgib = Perl::Gib->new();
 $perlgib->doc();
